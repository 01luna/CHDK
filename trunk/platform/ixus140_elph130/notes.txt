************************************
IXUS140 / ELPH130IS Notes 
************************************
To enter CHDK <ALT> mode use the [>] Play button.
This may be changed to Video in the options

This camera does not have an adjustable aperture. It does have an ND filter

This is an incomplete, pre-alpha port, use at your own risk!

TESTED
******
shutter, iso and ND override (not comprehensively)
llibtst.lua
ptp control, rec/play switching
RAW/DNG capture (has file/directory issues noted below)

NOT IMPLEMENTED
***************
dark frame control
usb remote hook
fast live view buffer
extra long exposures / exp_drv task
filewrite hook
led functions
video functions
exmem support (note, camera has ~1.7 mb regular heap free on startup with CHDK)

KNOWN BUGS
**********
Folder naming and file number for raw and get_image_dir have issues
Debug LED is AF LED!
Palette is wrong, not implemented yet
viewport dimensions are wrong, not implemented yet =histogram, zebra, ptp live view etc broken
DNG color matrix is copy+paste from D10
